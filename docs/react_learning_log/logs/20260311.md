# (2026-03-11) grid 구조 생성
<br/>
grid페이지 생성을 위한 페이지용 features 폴더 생성<br />
grid화면 자체가 ui적으로 매우 빈번하고 다양하게 사용되어 대부분 ui library를 사용해서 이뤄지는데<br />
전체적으로 굴러가는 모습을 보기위해서 가장 일반적인 구조를 잡아 진행함<br />
header, list. pagination 구조를 따로 두고 이것을 모두 합친 grid를 하나로 만드는 과정으로 진행.<br /><br />

각 구조의 컨포넌트 구성은 아래와 같음<br />
- header는 통상적으로 전체 목록 및 노출목록 select, button(이동)<br />
- list는 table구조이며 sample data로 던져질때 width 까지는 처리되도록 정의<br />
- pagination은 중앙 버튼으로 처음,끝, 이전, 다음 버튼까지 추가 prop로 페이지 갯수 값 전달<br />
